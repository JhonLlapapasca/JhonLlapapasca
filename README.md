- 🌐 **Desarrollo Frontend:** Experiencia en la construcción de interfaces dinámicas con **Vue.js** y **Nuxt.js** e integración con APIs REST.
- 📚 **Evolución Continua:** Enfocado en la adopción de buenas prácticas de desarrollo, arquitectura limpia y el aprendizaje de nuevas herramientas tecnológicas.
